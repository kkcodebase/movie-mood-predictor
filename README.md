# Movie Mood Predictor

An intelligent web application that analyzes user movie reviews to determine the overall sentiment (positive, negative, or neutral) using AWS cloud services.

---

## Features

- **Real-time Sentiment Analysis** — Uses AWS Lambda and API Gateway to analyze movie reviews instantly.  
- **Serverless Architecture** — Built with AWS Lambda, ensuring scalability and cost efficiency.  
- **Deployed on AWS S3** — Frontend hosted via Amazon S3 for static web hosting with integrated API backend.  

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (AWS Lambda)  
- **Cloud Services:** AWS API Gateway, Lambda, S3  
- **Deployment Tools:** AWS CLI, PowerShell
